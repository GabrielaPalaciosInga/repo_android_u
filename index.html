COMUNICACIÓN EN REDES LAN Y WAN LAN

LAN (Red de Área Local):

  • Cobertura: Áreas pequeñas (oficina, edificio).
  • Propósito: Conectar dispositivos cercanos para compartir recursos.

WAN (Red de Área Amplia):
  • Cobertura: Áreas pequeñas (oficina, edificio).
  • Propósito: Conectar dispositivos cercanos para compartir recursos.

Componentes Principales

LAN:

* Switches: Conectan dispositivos dentro de la red.
* Routers: Conectan la LAN con otras redes.
* Cables Ethernet: Conexiones físicas entre dispositivos.
* Puntos de Acceso Wi-Fi: Conexiones inalámbricas.

WAN:

* Routers: Dirigen el tráfico entre diferentes redes.
* Enlaces de Comunicación: Cables submarinos, líneas alquiladas, conexiones satelitales.
* Modems: Modulan y demodulan señales para transmisión a larga distancia.

Proceso de Comunicación

LAN:

Direcciones MAC: Identificación única de dispositivos.
Protocolos: Ethernet (IEEE 802.3): Protocolo estándar para redes cableadas.
Wi-Fi (IEEE 802.11): Protocolo estándar para redes inalámbricas.
CSMA/CD: Método para evitar colisiones en redes cableadas.
CSMA/CA: Método para evitar colisiones en redes inalámbricas.
Transmisión de Datos: Switches filtran y envían datos a dispositivos específicos.

Proceso de Comunicación

WAN:

Direcciones IP: Identificación única para enviar datos entre redes.
Protocolos:
TCP/IP: Protocolo principal para la transmisión de datos en Internet.
BGP (Border Gateway Protocol): Protocolo para enrutamiento entre diferentes sistemas autónomos.
MPLS (Multiprotocol Label Switching): Protocolo para gestión eficiente del tráfico.
VPN (Virtual Private Network): Protocolo para crear conexiones seguras.
Enrutamiento: Routers determinan la mejor ruta para los datos.
Fragmentación y Reensamblaje: Datos grandes se dividen en fragmentos pequeños y se reensamblan en el destino.

Diferencias Clave

Escala:

LAN: Áreas pequeñas.

WAN: Grandes distancias.

Velocidad y Latencia:

LAN: Alta velocidad, baja latencia.

WAN: Velocidades variables, mayor latencia.

Complejidad y Costo:

LAN: Menos costosa y compleja.

WAN: Más costosa y compleja.
